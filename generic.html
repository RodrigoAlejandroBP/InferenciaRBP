    mediana <- function(x) {
    ordenar <- sort(x)
    br>z <- length(ordenar)
    if((z%%2) == 0) {              //Compruebo si es par o impar
    resultado <- ((x[z/2]+x[(z/2)+1])/2)

    y <- (z+1)/2

  • Obtener la varianza

    • En teoría de probabilidad, la varianza o variancia (que suele representarse como {\displaystyle \sigma ^{2}} \sigma^2) de una variable aleatoria es una medida de dispersión. La varianza tiene como valor mínimo 0.Hay que tener en cuenta que la varianza puede verse muy influida por los valores atípicos y no se aconseja su uso cuando las distribuciones de las variables aleatorias tienen colas pesadas

    funcaux <- function(x){
	x1 = sort(x)
	aux <- 0
	for(i in 1:length(x)){
	  aux <- aux + x1[i]/length(x)
	}
	return (aux)
  }
  
  varianza <- function(vector){
	aux <- 0
	vector1 = sort(vector)
	for(i in 1:length(vector)){
	  aux <- aux + ((vector1[i] - funcaux(vector1))^2)
	}
	
	return (aux/(length(vector)-1) )
  }

    cuartiles

    Los cuartiles son valores que dividen una muestra de datos en cuatro partes iguales. Utilizando cuartiles puede evaluar rápidamente la dispersión y la tendencia central de un conjunto de datos, que son los pasos iniciales importantes para comprender sus datos. 25% de los datos es menor que o igual a este valor.

  • 1 Cuartil 

    • cuart25 <-function(x){
	alpha <- 0
	ordenada <- sort(x)
	alpha <- 0.25
	n <- length(ordenada)
	i <- alpha*(n+1)
	e <- floor(i)
	d <-i - e
	return((1-d)*ordenada[e]+((d)*ordenada[e+1]))
}

    Ejemplo del primer Cuartil

    cuart25(c(168,173,175,178,179,180,187))

    El cuartil 25 en este caso seria de :

    ## [1] 173

    Rango intercuartil

    En estadística descriptiva, se le llama rango intercuartílico o rango intercuartil, a la diferencia entre el tercer y el primer cuartil de una distribución. Es una medida de la dispersión estadística. Por lo mismo como ya se tienen las funciones anteriores es fácil resolverla.

    rangoint <- function(x){
	intercuart <- cuart75(x)-cuart25(x)
	return(intercuart)
	}
    Ejemplo
    rangoint(c(168,173,175,178,179,180,187))
    ## [1] 7

    Datos atipicos

    Un valor atípico es una observación extrañamente grande o pequeña. Los valores atípicos pueden tener un efecto desproporcionado en los resultados estadísticos, como la media, lo que puede conducir a interpretaciones engañosas.

  • Metodo de tukey

    • También llamado “método de las bisagras de Tukey”, consiste encrear intervalos en torno al primer y tercer cuartil tal que:
    *Barrera interior inferior: BII=Q1−1,5RI.
    *Barrera interior superior: BIS=Q3+ 1,5RI.
    *Barrera exterior inferior: BEI=Q1−3RI.
    *Barrera exterior superior: BES=Q3+ 3RI 

    tukey <- function(x){
	x1 <- sort(x)
	bii <- cuart25(x1)-(1.5*rangoint(x1))
	bis <- cuart75(x1)+(1.5*rangoint(x1))
	lista = c(bii,bis)
	return (lista)
}
    Ejemplo
    tukey(c(168,173,175,178,179,180,187))

    El rango de los atipicos esta entre -162.5 y +190.5

    ## [1] 162.5 190.5

  • Metodo de puntuacion Z

    • Si consideramos la regla empírica, sabemos que a proximadamenteel100 %de los datos está en el intervalo[x−3S;x+ 3S]. Es muy improbable que un dato esté fuera de este intervalo y, en caso queesto ocurriese, éste se llamaría undato atípico(o outlier).

    puntZ <- function(x){
	x1 <- 0
	prom <- 0
	des <- 0
	x1 <- sort(x)
	prom1 <- prom(x1)
	des <- desves(x1)
	lista <- c(0,0)
	lista[c(1)] <- prom1 - (3*des)
	lista[c(2)] <- prom1 + (3*des)
	return (lista)
}
    Ejemplo
    puntZ(c(168,173,175,178,179,180,187))

    El rango de los atipicos esta entre 159.1905 y +195.0952

    ## [1]159.1905 195.0952
